# Customer Insights Analytics Project

## Problem Statement

Understanding customer behavior is essential for businesses to make informed decisions and improve customer experience. This project aims to analyze customer data to identify patterns, trends, and key factors influencing customer behavior, enabling data-driven decision-making.

## Dataset

The dataset contains customer-related information such as demographics, transactions, and behavioral attributes. It was used to explore patterns and extract meaningful insights.

## Tools Used
Python – Data loading, cleaning, and exploratory data analysis
SQL (PostgreSQL) – Data querying and analytical operations
Power BI – Data visualization and dashboard creation
Gamma AI – Presentation (PPT) generation

## Python (Data Processing & EDA)
Loaded dataset and handled missing values
Performed data cleaning and preprocessing
Conducted exploratory data analysis (EDA)
Identified initial patterns and relationships in data

## SQL (Data Analysis)
Imported cleaned data into PostgreSQL
Wrote queries to analyze customer behavior
Performed aggregations, filtering, and joins
Extracted insights for business understanding

## Power BI (Dashboard)
Built an interactive dashboard
Visualized key metrics and trends
Represented customer behavior insights clearly
Enabled easy interpretation for stakeholders

## Key Insights
Identified major customer segments and trends
Observed patterns in customer behavior and engagement
Highlighted key factors influencing business performance
Provided insights to support data-driven decisions
📑 Report & Presentation
Created a detailed report summarizing analysis and findings
Developed a presentation using Gamma AI for clear communication of insights
